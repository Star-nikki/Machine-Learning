# Machine Learning Fundamentals

A practical repository containing Jupyter Notebook implementations for core **Machine Learning** concepts, focusing on **Data Preprocessing**, **Dimensionality Reduction**, and **Ensemble Learning**.

<br>

## ðŸ“‚ Repository Contents

This project is organized into three main areas, with all examples implemented in **Python** using **Jupyter Notebooks**.

### 1. Data Preprocessing & EDA

Notebooks covering essential steps to prepare data for model training.

* `Normalization.ipynb`: Demonstrates data **scaling and normalization** techniques.
* `One hot coading.ipynb`: Implementation of **One-Hot Encoding** for categorical features.
* `Outliers Detection.ipynb`: Methods for identifying and handling **outliers**.
* `Blueprint for classification.ipynb`: A complete **classification workflow** template.
* `placement.csv`: The primary **dataset** used across various examples.

### 2. Dimensionality Reduction

Notebooks focused on transforming high-dimensional data.

* `PCA.ipynb`: Implementation of **Principal Component Analysis**.
* `CCA analysis.ipynb`: Implementation of **Canonical Correlation Analysis**.

### 3. Ensemble Methods

Folders containing specific ensemble learning techniques.

* **Bagging/**: Examples of **Bootstrap Aggregating** (e.g., Random Forest).
* **Voting/**: Examples of **Voting Classifiers** (Hard and Soft Voting).

## ðŸš€ Getting Started

To run these notebooks locally, you will need **Python** and standard data science libraries (`pandas`, `numpy`, `scikit-learn`, `matplotlib`, `jupyter`).

1.  Clone the repository:
    ```bash
    git clone [https://github.com/Star-nikki/Machine-Learning.git](https://github.com/Star-nikki/Machine-Learning.git)
    ```
2.  Install dependencies (using `pip`):
    ```bash
    pip install pandas numpy scikit-learn matplotlib jupyter
    ```
3.  Launch Jupyter and explore the notebooks!
    ```bash
    jupyter notebook
    ```
